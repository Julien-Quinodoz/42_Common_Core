# 42 Common Core

**Mon parcours à travers le tronc commun de l'école 42**

---

## À propos

Ce dépôt contient l'ensemble de mes projets réalisés durant le cursus **Common Core** de 42. Chaque projet explore des concepts fondamentaux en informatique, de la programmation bas niveau à l'architecture système avancée.

Le programme 42 est conçu pour développer l'autonomie, la résolution de problèmes et la collaboration par le biais de l'apprentissage entre pairs.

## Compétences développées

Au fil de ces projets, j'ai acquis des compétences dans :

- Programmation bas niveau : gestion de la mémoire, pointeurs, appels système
- Algorithmes et structures de données : tri, parcours de graphes, optimisation
- Systèmes d'exploitation : gestion des processus, threading, synchronisation
- Réseau : TCP/IP, protocole HTTP, architecture client-serveur
- Programmation orientée objet : encapsulation, héritage, polymorphisme
- Programmation graphique : raycasting, gestion d'événements, fenêtres
- Administration système : machines virtuelles, Linux, sécurité
